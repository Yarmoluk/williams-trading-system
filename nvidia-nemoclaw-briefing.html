What Is OpenClaw?

OpenClaw is the open-source agent runtime — the platform where AI agents ("claws") live and execute tasks. Think of it as the operating system for AI agents. It runs on your local machine, not in the cloud.

It became the fastest-growing open-source project ever. Every developer building AI agents is using it or something like it.

What Is NemoClaw?

NVIDIA's enterprise wrapper around OpenClaw. One-command install that adds:

  • 1OpenShell Runtime — sandboxed execution environment. Agents can't leak your data.
  • 2Privacy Router — agents use local Nemotron models AND cloud frontier models, with policy controls on what goes where.
  • 3Security Guardrails — least-privilege access, network limits, policy enforcement.
  • 4Runs on — GeForce RTX, RTX PRO, DGX Spark, DGX Station.
